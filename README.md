# Todo App (Vanilla JS)

A simple, lightweight Todo application built with plain HTML, CSS, and JavaScript. No frameworks or build tools required.

## Features

- Add new todos via a modal dialog
- Search todos by text
- Filter by status (All / Done / Pending)
- Mark todos as done
- Delete todos
- Data persisted in `localStorage`

## Running

No installation needed. Open `index.html` in any modern browser:

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Or serve locally:

```bash
npx serve .
```

Then open [http://localhost:3000](http://localhost:3000).

## Prerequisites

- Any modern web browser (Chrome, Firefox, Edge, Safari)
